# 🦷 Sorriso + Odontologia

## 💻 Sobre o Projeto

O **Sorriso + Odonto** é uma Landing Page moderna, elegante e responsiva desenvolvida com foco em **SEO (Search Engine Optimization)** e boas práticas de Front-end. O projeto simula a página de uma clínica odontológica, apresentando especialidades, corpo clínico, depoimentos de pacientes e canais de agendamento.

Este projeto foi desenvolvido como parte do currículo do curso **Técnico em Desenvolvimento de Sistemas** da Escola **SENAI A. Jacob Lafer**.

---

## ⚙️ Funcionalidades e Conceitos Aplicados

Durante o desenvolvimento, foram aplicados os seguintes conceitos técnicos:

* **HTML Semântico:** Uso correto de tags (`<header>`, `<main>`, `<section>`, `<article>`) para melhorar a acessibilidade e o ranqueamento nos motores de busca (SEO).
* **CSS Moderno:**
  * Layouts flexíveis e dinâmicos com **Flexbox**.
  * Posicionamento, paleta de cores clean (voltada para a área da saúde) e estilização de elementos.
  * Responsividade com **Media Queries** focando na filosofia *Mobile First*.
* **JavaScript Básico:** Manipulação do DOM para interatividade do menu responsivo (Efeito Toggle) e transições de elementos.
* **Versionamento:** Controle de versão utilizando **Git** e **GitHub Desktop**.
